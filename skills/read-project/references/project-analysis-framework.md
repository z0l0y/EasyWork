# 项目阅读分析维度

> 用于 read-project 技能 · 结构化项目分析

## 5 维度扫描法

| 维度 | 关注点 | 方法 |
|------|--------|------|
| **架构** | 分层/模块划分/依赖方向 | 读 CLAUDE.md + 顶级目录结构 |
| **技术栈** | 语言/框架/数据库/中间件 | 读 package.json / Cargo.toml / requirements.txt |
| **数据流** | 输入→处理→输出路径 | 追踪一个典型请求的完整链路 |
| **代码模式** | 命名约定/错误处理/测试风格 | 抽样 3-5 个核心模块 |
| **工程实践** | CI/CD/代码审查/发布流程 | 读 .github/ + Makefile |
