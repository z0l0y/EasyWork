# 论文阅读框架

> 用于 read-paper 技能 · 结构化论文分析

## 三遍阅读法

### 第一遍（5-10 分钟）：快速筛选
1. 读标题 + 摘要 + 结论
2. 扫一眼图表（了解主要结果）
3. 决定：值得深读 / 存档备查 / 跳过

### 第二遍（30-60 分钟）：理解大意
1. 读引言 → 了解背景和贡献
2. 读方法概述 → 不深究细节
3. 读实验结果 → 关注对比基准
4. 能用一句话总结：这篇论文做了什么、效果如何

### 第三遍（1-3 小时）：深度理解
1. 重现关键推导（公式/算法）
2. 质疑实验设计（是否有遗漏基准？是否有作弊嫌疑？）
3. 思考可复用性（这个方法能否用在我的项目里？）
