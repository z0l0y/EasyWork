# 快问快答模式参考

> 用于 quick-answer 技能 · 答案结构优化

## 输出结构金字塔

```
         TL;DR (1 句话)
        /              \
   核心要点 (3-5 条)   代码/命令 (直接可用)
        \              /
         延伸阅读 (可选)
```

## 答案质量检查

- [ ] TL;DR 是否 ≤ 50 字？
- [ ] 核心要点是否按重要性排序？
- [ ] 代码示例是否可直接复制运行？
- [ ] 是否避免了"参考文档 X"式的转发性回答？
- [ ] 答案是否自包含（不依赖上下文理解）？
